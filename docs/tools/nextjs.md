### Overview {#overview}

<div style={{ border: '1px solid #e2e8f0', borderLeft: '4px solid #3b82f6', borderRadius: '8px', padding: '20px', backgroundColor: '#f8fafc', marginBottom: '32px' }}>
  <p style={{ fontSize: '14px', color: '#475569', margin: '0', lineHeight: '1.7' }}>Next.js serves as AquaTrack's frontend framework, delivering server-rendered pages for fast initial loads and client-side interactivity for real-time dashboards. With App Router, React Server Components, and built-in optimizations, it provides the foundation for every user-facing interface -- from the customer self-service portal to internal operator dashboards.</p>
</div>
