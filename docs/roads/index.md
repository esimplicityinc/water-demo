# Road Items

All roadmap items with governance tracking.

## By Phase

- **Phase 0: Foundation** (Complete) - Core infrastructure and setup
- **Phase 1: Bot Identity** (Complete) - Bot registration and identity management
- **Phase 2: Token & Promise** (In Progress) - Token economics and promise lifecycle
- **Phase 3: Settlement & Advanced** (Proposed) - Settlement logic and advanced features

## Governance Status

Each road item includes governance tracking:

- **ADRs (Architecture Decision Records)** - Validated before implementation
- **BDD Scenarios** - Feature files and test coverage
- **NFRs (Non-Functional Requirements)** - Performance, security, and accessibility
